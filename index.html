Chessboard


Please enter width, height of the chessboard and symbol for fields representation and the app will draw it.

Note: All 3 fields below should be filled. Width and heigth value should be positive EVEN integers between 1 and 50. Only one-char symbols are allowed (* - GOOD, ** - BAD).

Envelopes Analisys


We have 2 envelopes with the sides (a, b) and (c, d). Please enter length of each mentioned side and the app will find out if it is possible to put one envelope into another.

Note: All 4 fields below should be filled. Only positive numbers up to 50 can be used as length values of envelopes sides.

Triangles Sorting *works only with 3 triangles


Please fill all the fields below with names of the triangles and each side length and the app will sort them by the area (from the biggest to the smallest).

Note: All the fields below should be filled. Only positive numbers can be used as length value for triangles sides. Each triangle should be valid: length of each side should not be greater than the sum of two other sides

Palindrome


Please enter any number and the app will find palindrome inside it.

Note: Only integers are allowed. Number should contain of 100 digits or less.

Lucky Tickets


There are two methods of defining lucky tickets: Simple and Complex. Please enter min and max ticket numbers and the app will count number of lucky tickets using both methods and find the winner.

Note: Only integers between 0 and 999999 are allowed.

Number Sequence


Please enter length of the number sequence and the check number below.

Note: Only positive integers are allowed. Length should be between 1 and 100, the check number should be between 1 and 1000000.

Fibonacci Sequence


Please enter min and max numbers OR length of the sequence and the app will find Fibonacci number sequence in mentioned range.

Note: Only positive integers are allowed. Length value should be between 1 and 100, min and max numbers should be between 1 and 1000000. Max number should be greater than min number
